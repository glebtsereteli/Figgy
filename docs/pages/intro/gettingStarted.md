# Getting Started

## Installation
1. Download the latest `.yymps` package from the [Releases](https://github.com/glebtsereteli/Figgy/releases) page.
2. Import the package into your project.
    * Drag'n'drop the `Figgy vX.X.X.yymps` file into your project.
    * OR import it via Tools/Import Local Package.
3. Click Add All, then Import.
